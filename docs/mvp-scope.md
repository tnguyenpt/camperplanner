# MVP Scope Summary

## Primary user
- Initial: solo planner
- Future: group planning with shared visibility and coordination

## MVP objectives
- Organize trip planning into clear phases
- Help decide between multiple campsite options
- Keep itinerary and trip logistics in one place
- Surface overall planning progress at a glance

## MVP feature areas
1. Trip basics
- Fields: name, location, start date, end date, notes, status
- Invitee status summary: accepted, declined, pending

2. Campsite decision phase
- Add campsite candidates to a trip
- Set per-candidate status: `unsearched`, `searching`, `booked`, `rejected`
- Add upvote/downvote counts to compare options
- Mark one candidate as selected when booked

3. Itinerary planning phase
- Add day plans and notes
- Reorder itinerary entries with up/down controls (drag-and-drop deferred)
- Mark itinerary items complete/incomplete

4. Progress tracking
- Trip-level progress metric from planning milestones
- Visual status for planning phase completion

## Deferred to v2
- Real multi-user collaboration and auth
- Campflare-style monitoring and external alerts
- Weather integrations
- Drag-and-drop planning interactions
